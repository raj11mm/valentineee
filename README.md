# 💖 Valentine Proposal Web App

An interactive and fun romantic web application built using HTML, CSS, and JavaScript. This project simulates a playful proposal experience with animations, floating hearts, confetti effects, and dynamic UI interactions.

---

## 💡 Features

- Interactive YES / NO proposal buttons  
- “NO” button that moves away on hover 😄  
- Animated floating hearts & sparkles  
- Confetti celebration effect 🎉  
- Multi-screen flow (Proposal → Agreement → Celebration)  
- Click-based heart animations  
- Background music support  

---

## 🛠️ Tech Stack

- HTML  
- CSS (Animations + Responsive Design)  
- JavaScript (DOM Manipulation + Events)

---

## 🎯 What I Learned

- DOM manipulation and event handling  
- CSS animations and keyframes  
- Creating interactive UI experiences  
- Managing multi-screen UI flow  
- Basic animation timing and effects
